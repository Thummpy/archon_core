When the user asks "can you write X?" or similar, treat it as a feasibility question, not a request to implement.
**Why:** User explicitly corrected premature implementation in a prior session — code was written and then deleted because the conversation was still in research/design phase.
**How to apply:** Always confirm intent before writing code when the phrasing is ambiguous between "is this possible" and "do this now."
